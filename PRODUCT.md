# Product

## Register

product

## Users

Software developers and team leads reviewing GitHub pull requests. They work in terminals and code editors, often late at night, and want speed, precision, and trustworthiness — not decoration.

## Product Purpose

Insightor is an AI-powered PR review console. It analyzes code changes across multiple PRs simultaneously, surfaces findings by severity, and publishes reviews to GitHub. Success means reviewers catch more issues in less time, with confidence in the tool's signal quality.

## Brand Personality

Professional, precise, focused. Three words: quiet confidence, developer-native. The tool should feel like it belongs next to a terminal and a code editor — familiar to anyone who uses Linear, Raycast, or Stripe's dashboard.

## Anti-references

- **Not** a flashy SaaS dashboard with hero metrics and gradient CTAs
- **Not** a consumer app with playful illustrations and rounded everything
- **Not** a marketing page with scroll-driven animations and display typography
- **Not** a "modern AI tool" with purple-blue gradients and glassmorphism

## Design Principles

1. **Clarity over decoration** — every visual element must earn its place by conveying information or affordance
2. **Speed as a feature** — the interface must feel instant; motion serves state change, not choreography
3. **Consistency is trust** — the same component must look and behave the same way everywhere
4. **Developer-native** — density when needed, monospace where appropriate, keyboard-first interactions
5. **Dark by default** — the tool lives in a developer's world of dark terminals and editors

## Accessibility & Inclusion

- WCAG 2.1 AA minimum for all text and interactive elements
- Full keyboard navigation (all actions reachable without a mouse)
- Respect `prefers-reduced-motion` — all animations must degrade gracefully
- Touch targets ≥ 44×44px on all interactive elements
